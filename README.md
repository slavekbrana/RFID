# RFID SmartLog - Docházkový systém

Jednoduchý docházkový systém postavený na Pythonu (Flask) a RFID čtečce.

## Funkce
- Evidence příchodů a odchodů pomocí RFID čipů.
- Správa uživatelů (přidávání/mazání).
- Moderní webové rozhraní (Dark Mode).
- Ukládání dat do CSV souborů.

## Jak spustit
1. Nainstalujte Python 3.x.
2. Nainstalujte Flask: `pip install flask`.
3. Spusťte aplikaci: `python app.py`.
4. Otevřete v prohlížeči `http://127.0.0.1:5000`.
